-Place the Python folder in desktop.
-Launch Python shell.
-Open via python shell "nmea to mysql" file.
-Roll down in the code until you reach the changing password note, and change it to your password 
as told there as well.
-Launch MySql, but dont enter the user yet (root).
-Go back to python and run "nmea to mysql" code after the password have been changed.
-Type 1 to import the nmea file
-Enter the filename.txt (asi.txt for example)
-Go back to MySql, enter your user. You will see new scheme called nmea, double click on it,
and you will see the tables.

*if you don't see the new Scheme, exit and enter again, it will appear.
